# NoctOS Implementation Plan

## Overview
Build a fully functional macOS-inspired WebOS called "NoctOS" as a single `index.html` file. Pure vanilla HTML/CSS/JS, zero dependencies. Then create 3 devlogs, commit, and push to GitHub.

**Repo**: `https://github.com/kaorii-ako/noctOS.git` (empty, no commits)
**Hackatime**: `https://hackatime.hackclub.com/my/projects`

---

## New Features (beyond the guide spec)
The user wants ALL of these added:
1. **Notification Center** — Bell icon in menu bar → slide-in panel from right with system notifications, weather widget (mock), calendar widget (mock)
2. **Mission Control** — Ctrl+Up zooms out and tiles all open windows
3. **Screensaver** — After 90s idle, full-screen floating clock animation
4. **Dock Right-Click Context Menu** — Open/Hide/Quit per app
5. **Calendar Widget** — Dropdown from menu bar clock showing a mini calendar

---

## Build Phases

### Phase 1: Foundation Shell
**Write to**: `index.html`

1. HTML skeleton + all CSS custom properties (`:root` vars)
2. Global reset (`*` box-sizing, body setup)
3. **Boot sequence**: 4-phase animation (moon logo fade → "NoctOS" text → progress bar → fade out)
4. **Menu bar**: Fixed 28px, z-index 9999
   - Left: NoctOS moon logo, app name, File/Edit/View/Window/Help dropdowns
   - Right: Battery SVG + "87%", WiFi SVG, Volume SVG, live clock (updates every second), Search icon
5. **Desktop**: Full viewport, 3 desktop icons (Macintosh HD, Projects, README.txt), right-click context menu (New Note, New Folder, Change Wallpaper, About NoctOS), wallpaper cycling (base → nebula → aurora → midnight)
6. **Dock**: Pill container, 7 icons (Finder/Safari/Terminal/Notes/Calculator/Settings/Trash) with colored SVG icons
   - Magnification: `scale = 1 + (maxScale - 1) * Math.max(0, 1 - distance / radius)`, maxScale=1.6, radius=100px
   - Hover labels, running indicators (2px dot), bounce animation on click

**Verification**: Open index.html → boot plays → menu bar has live clock → desktop icons visible → dock magnifies smoothly

### Phase 2: Window System
**Add to**: `index.html`

1. `NoctOS.WindowManager` namespace — create/focus/close/minimize/restore/toggleFullscreen
2. Window HTML structure: title bar (40px, draggable), traffic lights (red/yellow/green with hover icons), title, content area, resize handle
3. **Drag**: titlebar mousedown → mousemove tracking → snap zones (top=fullscreen, left=half, right=half) with ghost preview
4. **Resize**: bottom-right handle, debounced
5. **Traffic lights**: red=close, yellow=minimize, green=fullscreen
6. **Z-index**: base 100, increment on focus, cap 9990
7. **Animations**: open (scale 0.95→1, opacity 0→1, 180ms), close (120ms), minimize (200ms)

**Verification**: Windows can be created, dragged, resized, minimized, closed, fullscreened, and stacked

### Phase 3: Core Apps
**Add to**: `index.html`

1. **Finder** (760×480): sidebar + content, grid/list toggle, breadcrumb, 2-level deep fake file system, double-click navigation
2. **Terminal** (600×400): colored prompt, command input, 15+ built-in commands (help, ls, cd, pwd, whoami, date, echo, clear, history, open, neofetch, cat, rm, sudo, exit), tab autocomplete, up/down history
3. **Notes** (680×480): sidebar note list, contenteditable editor, toolbar (bold/italic/underline/list/heading), 3 pre-loaded notes, localStorage persistence, debounced auto-save (800ms)

**Verification**: Each app opens from dock, internal features work (terminal commands execute, notes save/load, finder navigates)

### Phase 4: Advanced Apps
**Add to**: `index.html`

1. **Calculator** (280×380): display + 4×5 button grid, full math logic, keyboard support, circular buttons
2. **Settings** (680×460): sidebar panels (General/Appearance/Wallpaper/Dock/About), live accent color picker (6 colors), wallpaper switching, dock settings, About panel mimicking macOS
3. **Safari** (900×560): tab bar, URL bar, favorites bar, start page with search, iframe loading, loading progress bar
4. **Spotlight** (z-index 9988): Cmd+Space trigger, 620px panel, search input, results by category, keyboard navigation
5. **Keyboard shortcuts**: Cmd+W (close), Cmd+M (minimize), Cmd+H (hide), Cmd+Tab (app switcher), Escape (dismiss)

**Verification**: Calculator works, settings accent changes globally, Safari loads URLs, Spotlight finds apps, shortcuts work

### Phase 5: New Features + Stretch Goals
**Add to**: `index.html`

1. **Notification Center**: Bell icon in menu bar → slide-in panel from right
   - System notifications (mock): "Welcome to NoctOS", "System update available", "Backup complete"
   - Weather widget (mock data): temperature, condition, icon
   - Calendar widget: mini month view with today highlighted
2. **Mission Control**: Ctrl+Up → zoom out effect, tile all open windows in a grid
3. **Screensaver**: After 90s idle → full-screen with large drifting clock text
4. **Dock right-click**: Context menu with Open/Hide/Quit per app
5. **Calendar dropdown**: Click clock in menu bar → mini calendar dropdown

**Verification**: Notification center slides in/out, Mission Control tiles windows, screensaver triggers after idle, dock right-click works

5. Add devlogs + commit: "Add devlogs"
6. `git push origin main`
